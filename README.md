# 🛡️ SOC Command — مركز العمليات الأمنية

لوحة تحكم تفاعلية (Dashboard) لمركز عمليات أمنية (Security Operations Center) تعرض بيانات ومؤشرات لحظية حول حالة التهديدات الأمنية داخل المؤسسة.

🔗 **رابط العرض المباشر:** [nagissoi81-stack.github.io/DashBordSoc](https://nagissoi81-stack.github.io/DashBordSoc/)

---

## 📋 نظرة عامة

تم تصميم هذه اللوحة لمحاكاة بيئة عمل حقيقية لمحلل أمن سيبراني (SOC Analyst)، حيث تعرض المؤشرات الرئيسية للأداء (KPIs) المستخدمة في مراقبة وتحليل الحوادث الأمنية بشكل لحظي.

## ✨ المزايا الرئيسية

- **حالة المؤسسة اللحظية** — مؤشر مباشر لمستوى التهديد العام مع ساعة وقت حي.
- **عدد التنبيهات اليومية** — تتبع لحظي لعدد التنبيهات مقارنة باليوم السابق.
- **التنبيهات الحرجة** — تنبيهات تتطلب تدخلاً فورياً من فريق الاستجابة.
- **الأجهزة المصابة** — نسبة الأجهزة المتأثرة من إجمالي أجهزة الشبكة.
- **متوسط زمن الاستجابة (MTTR)** — قياس الأداء مقارنة بحد اتفاقية مستوى الخدمة (SLA).
- **الحوادث المفتوحة والمغلقة** — رسم بياني يوضح نسبة الحوادث قيد المعالجة مقابل المغلقة.
- **الهجمات حسب النوع (24H)** — تصنيف الهجمات خلال آخر 24 ساعة.
- **المستخدمون الأكثر تعرضاً للهجمات** — قائمة بأعلى 5 مستخدمين استهدافاً.
- **خريطة مصدر الهجمات (GEO-IP)** — تحديد جغرافي لمصادر الهجمات.
- **مصفوفة MITRE ATT&CK** — خريطة حرارية (Heatmap) للتكتيكات والتقنيات المستخدمة في الهجمات، مبنية على إطار عمل MITRE ATT&CK المعتمد عالمياً.
